# Sistema de Gestão de Projetos

## Descrição do Projeto

Este projeto está sendo desenvolvido durante processo seletivo para a Mega Júnior em 2026. Tem como objetivo solucionar os problemas de gerenciamento de projeto e gestão de membros da empresa para garantir rastreabilidade das atividades desenvolvidas e carga de trabalho ótima para os funcionários da Mega Júnior. A solução apresentada é um MVP com um protótipo web onde os usuários devem ser capazes de criar, gerenciar e visualizar projetos e seus membros.

## Requisitos (Desenvolvedores)

Este sistema apresenta os seguintes requisitos para desenvolvimento : 

1. Docker (Versão >= 29.2.1)

Caso queira utilizar o sistema sem utilizar contêineres docker : 

1. NPM (Versão >= 12.12.1)
2. Node.JS (Versão >= 24.13.1)
